#####  Выбрать стартовую клетку
- Выбери любую клетку (например, `(0, 0)`).
- Отметь её как посещённую + добавить соседей
##### Инициализировать границу (Frontier)
- Найди все допустимые соседние клетки стартовой клетки через стены.
- Для каждой такой стены добавь её в список **границы**.  
    _Формат записи:_ координаты двух точек.

##### Основной цикл (пока граница не пуста)
- В ыбрать случайную стену из границы
-  Извлечь случайный элемент из списка bind.
-  Если не посещен уже, сломать стену между двумя
-  Добавить всех соседей, если индексы не выходят за пределы массивов и они ещё не посещены.
##### Шаг 4. Конец
Когда массив bind пуст — **все клетки посещены**, лабиринт сгенерирован.

Можно усложнить пространственную сложность переписав на рекурсию.
