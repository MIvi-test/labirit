# labirit

## О проекте

`labirit` — исследовательская программа для генерации, анализа и тестирования лабиринтов на языке C. Проект позволяет сравнивать несколько алгоритмов генерации и собирать метрики по времени, топологии и структуре.

## Поддерживаемые алгоритмы

- `prim` — алгоритм Прима для генерации минимального остовного дерева
- `growing_tree` — алгоритм «растущего дерева»
- `watson` — алгоритм Уотсона
- `dfs` — генерация на основе поиска в глубину
- `binary_tree` — алгоритм двоичного дерева
- `recursive_division` — рекурсивное разделение

## Что делает

- генерирует лабиринты разными алгоритмами;
- выполняет замеры производительности и анализ свойств лабиринта;
- сохраняет результаты в JSON-файлы;
- при запуске `./main --dump` выводит JSON-результаты в консоль.

## Структура проекта

- `backend/` — реализация алгоритмов генерации и вспомогательных функций;
- `frontend/` — визуализация / обход графа лабиринта;
- `analisys/` — инструменты для запуска бенчмарков и сохранения результатов;
- `tests/` — модульные и интеграционные тесты алгоритмов;
- `main.c` — главный исполняемый файл для запуска генерации и анализа;
- `json_parser.c`, `json_parser.h` — формирование и вывод JSON.

## Быстрый старт

Сборка основного исполняемого файла:

```bash
make compile
```

Запуск программы:

```bash
./main
```

Запуск с выводом JSON-результатов в консоль:

```bash
./main --dump
```

Результаты сохраняются в каталоге `analisys/json_data`.

## Тестирование

Сборка и запуск тестов:

```bash
make test
```

Файл тестов: `tests/test_algorithms.c`

Тесты проверяют:

- симметрию стен между соседними клетками;
- правильное оформление внешних границ;
- связность лабиринта (все клетки достижимы из начальной точки).

## Запуск бенчмарков

Сборка и запуск набора бенчмарков:

```bash
make run-benchmarks
```

## Полезные цели Makefile

- `make compile` — собрать `main`
- `make test` — собрать и запустить тестовый исполняемый файл `tests_run`
- `make run` — собрать и запустить `main`, затем `main --dump`
- `make run-benchmarks` — собрать и запустить `benchmarks`
- `make clean` — удалить собранные бинарники

## Требования

- компилятор C с поддержкой стандарта `C23`
- `gcc`, `clang` или совместимый компилятор
- библиотека math: `-lm`

## Контакты

Проект рассчитан на исследование алгоритмов генерации лабиринтов и может служить базой для дальнейшего расширения и визуализации.
